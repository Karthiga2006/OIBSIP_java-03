# OIBSIP_java-03
The MCQ Based Online Examination System is a console-based Java application designed to simulate an online multiple-choice examination environment. This project allows users to answer a set of predefined MCQ questions, evaluates their responses, calculates the final score, and displays the result.
The system is developed using core Java concepts and is suitable for academic mini-projects, internship submissions, and beginner-to-intermediate Java practice. It provides a clear understanding of how online exams function in a simplified, command-line format.
